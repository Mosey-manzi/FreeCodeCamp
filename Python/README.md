# Email Simulator (Python)

## Description
This project is part of my freeCodeCamp learning journey.  
It simulates a simple email system where users can send, read, and manage emails using Python.

## Features
- Send emails
- Store emails in a list
- Mark emails as read
- Display inbox messages

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## What I Learned
- Classes and objects
- Lists and indexing
- Methods and attributes
- Basic input validation
